This project is a Python-based QR Code Generator that creates customized QR codes with an embedded logo image. It uses the qrcode library to generate the QR code and Pillow (PIL) for image editing and logo placement.
Features


Generates QR codes from URLs
Embeds a logo in the center
Custom QR code colors
Saves the final QR image as a PNG file
Language: Python
Libraries Used:
qrcode
Pillow (PIL)
